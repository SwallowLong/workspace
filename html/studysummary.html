学习路线

  • 第一阶段：先学好web三剑客 使用Git
  • 基础篇：《精通css》、《css世界》、《js高级程序设计》、《你不知道的js》、《js语言精粹》
  • 网络篇：MDN、freeCodeCamp、Git
  • 使用cd进入文件夹

HTML

  • 单选按钮复选框中的name要一致 id不一致
    • li{新属性header、main、nav、section、article、footer}

CSS

  • border-radius:50%;可以将块变为圆形
  • 居中的方法
    1. position:absolute;top、bottom、left、right设置为0;margin:auto
    2. position:absolute;top:50%;left:50%;margin:-50%的width、heigth
  • transform:scale()可以用来扩大或缩小元素的比例
  • CSS实现动画效果animation
    1. animation-name属性设置动画的名称 @keyframes用来设置动画各阶段的变化 也是animation的名称
    2. animation-duration属性用来设置动画花费的时间
    3. animation-iteration-count属性用来设置动画循环的次数（其中infinite为无限循环）
    1. font属性中必须要有的为size和family且顺序就为这个

Node.js

  • node进入命令交互模式 control+c退出node模式
  • sudo代表超级权限
  • node运行时可以访问到 关闭时则访问不到本地自创服务器

Git和GitHub

  • git和远程库之间的联系
    1. 将目录变为Git可以管理的仓库：git init
    2. 在GitHub上插件一个new repo
    3. 关联并完成第一次推送 关联方法有两种：
      • https的方式:git remote add origin https://github.com/your-name/repository.git
      • ssh key方式:git remote add origin git@github.com:your-name/repository.git
        • li{如何生成ssh：ssh-keygen -t rsa -C "自己邮箱" 3个回车 cat ~/.ssh/id_rsa.pub(可以查看ssh) 然后复制ssh粘贴到GitHub即可}
      • push提示错误fetch 说明远程分支和本地有分歧 可以县合并 or 强制覆盖：git push origin master -f
  • 终端快捷键：
    1. 进入文件夹cd 退出文件夹cd ..
    2. !:wq为退出vim文件
  • vscode快捷键：
    1. shift+option+f键缩进快捷键
    2. command+➡️到此行结尾 command+⬅️到此行开头 command+⬆️到文档开头 command+⬇️到文档结尾
    3. command+b侧边栏的缩放
    4. command+数字键开页面数
